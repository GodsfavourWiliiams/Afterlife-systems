# Afterlife Systems (Next.js)

This project is now a Next.js App Router application.

## Requirements

- Node.js 20+

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Start production server:

```bash
npm run start
```

## Routes

- `/` - Home
- `/products` - Products
- `/about` - About
- `/contact` - Contact
- `/privacy` - Privacy Policy
- `/terms` - Terms of Service
